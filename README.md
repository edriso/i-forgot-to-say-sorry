# I forgot to say SORRY 🙏

A simple web app to create and download apology images with draggable stickers.

## Usage

1. Open [I forgot to say SORRY](https://edriso.github.io/I-forgot-to-say-sorry) in your browser
2. Click the title to customize it (e.g., "I forgot to say THANK YOU")
3. Write your apology in the text area
4. Add stickers by clicking on emojis — drag them to position (works on mobile too)
5. Click "Generate Apology Image"
6. The image will download automatically
7. Send the image to the person you want to apologize to

## Files

- `index.html` - Main HTML file
- `assets/style.css` - Styling with rem units and responsive design
- `assets/script.js` - Sticker management, drag & drop, and image generation
